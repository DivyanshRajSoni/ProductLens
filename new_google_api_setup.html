Get New Google Custom Search API Key

Get another 100 FREE searches per day by creating a new API key

Current Situation

Your current API key has reached its daily quota of 100 searches.

Solution: Create a new Google Cloud Project and get a fresh API key with another 100 free searches/day!

Step-by-Step Guide

  1. Go to Google Cloud Console
    Visit: https://console.cloud.google.com/
    Sign in with your Google account
  2. Create a NEW Project
    • Click on the project dropdown at the top
    • Click "NEW PROJECT"
    • Name it something like: ProductLens-2 or ImageSearch-New
    • Click "CREATE"
    • Wait for the project to be created and make sure it's selected
  3. Enable Custom Search API
    • Visit: Custom Search API Library
    • Make sure your NEW project is selected at the top
    • Click "ENABLE"
    • Wait for it to be enabled
  4. Create API Key
    • Go to: API Credentials
    • Click "+ CREATE CREDENTIALS" at the top
    • Select "API key"
    • Your new API key will be created!
    • COPY THE API KEY - You'll need this!
  5. Restrict API Key (Optional but Recommended)
    • Click on the API key you just created
    • Under "API restrictions", choose "Restrict key"
    • Select "Custom Search API"
    • Click "SAVE"
  6. Create/Use Custom Search Engine (CSE)
    • You can use your existing CSE ID: 2143768bfb9494f35
    • OR create a new one at: Create CSE
    • If creating new:
      • Select "Search the entire web"
      • Enable "Image search"
      • Click Create and copy the new CSE ID

After Getting Your New API Key

Update your code in these 3 places:

1. Update .env file:

GOOGLE_SEARCH_API_KEY=YOUR_NEW_API_KEY_HERE GOOGLE_SEARCH_ENGINE_ID=2143768bfb9494f35

2. Update ai_studio_code (6).html (around line 340):

var googleApiKey = 'YOUR_NEW_API_KEY_HERE'; var googleSearchEngineId = '2143768bfb9494f35';

3. Update api_diagnostic.html (around line 89-91):

const API_KEY = 'YOUR_NEW_API_KEY_HERE'; const CSE_ID = '2143768bfb9494f35';
IMPORTANT: Replace YOUR_NEW_API_KEY_HERE with the actual API key you copied from Google Cloud Console!

Pro Tips

  • Each Google account gets 100 FREE searches per day per project
  • You can create multiple projects for more quota
  • The quota resets at midnight Pacific Time (PT)
  • Keep your API keys secure and don't share them publicly
  • You can check your usage at: Check Quota
