Common Issues & Solutions

Most Common Problem:

Your Custom Search Engine (CSE) is not configured for "Image Search"

The CSE needs to be set to search the entire web and have Image Search enabled.

How to Fix:

  1. Go to Google Programmable Search Engine Console
  2. Find your search engine with ID:
    If this CSE doesn't exist, you'll need to create a new one.
  3. Click on the search engine to open settings
  4. In "Sites to search" section:
    • Remove any specific websites
    • Enable "Search the entire web"
  5. In "Image Search" section:
    • Turn ON the toggle for "Enable image search"
  6. Click "Update" to save changes
  7. Wait 2-3 minutes for changes to propagate, then test again on this page

Create New Custom Search Engine (If Needed)

If your CSE doesn't exist or you want to create a fresh one:

  1. Visit Create Programmable Search Engine
  2. Enter a name like "ProductLens Image Search"
  3. Under "What to search?":
    • Select "Search the entire web"
  4. Under "Settings":
    • Enable "Image search"
    • Enable "SafeSearch" (recommended)
  5. Click "Create"
  6. Copy the new Search engine ID
  7. Update your .env file with the new ID: 
    GOOGLE_SEARCH_ENGINE_ID=your_new_cse_id_here
  8. Also update ai_studio_code (6).html at line 341 with the new CSE ID

API Limits

Google Custom Search API provides 100 free searches per day.
After that, you'll need to upgrade to a paid plan or wait until the next day.
