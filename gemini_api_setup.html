Troubleshooting

Problem: "API key not valid" error
  • Make sure you copied the entire API key
  • Check for extra spaces before/after the key
  • Verify the key is between single quotes: 'YOUR_KEY'
  • Make sure you saved the file after pasting
Problem: "Quota exceeded" error
  • You've reached the free tier limit (1,500 requests/day)
  • Wait 24 hours for quota to reset
  • Or create a new API key with a different Google account
Problem: Can't access Google AI Studio
  • Make sure you're logged into your Google account
  • Try using Chrome browser
  • Clear browser cache and cookies
  • Try incognito/private mode
