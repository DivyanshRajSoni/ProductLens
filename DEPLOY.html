Method 2: Git Commands (For Developers)

3 minutes

Prerequisites:

  • Git installed on your computer
  • GitHub account created

Quick Commands:

# Navigate to project folder
cd "C:\Users\lenovo\OneDrive\Desktop\my_project"

# Initialize git (if not already done)
git init

# Add all files
git add .

# Check what will be committed (optional)
git status

# Commit
git commit -m "Initial commit - ProductLens ready for deployment"

# Create repository on GitHub first, then:
# Replace YOUR-USERNAME with your GitHub username
git remote add origin https://github.com/YOUR-USERNAME/productlens.git

# Push to GitHub
git push -u origin main
Note: Agar git push fail ho jaye:
  • GitHub pe repository pehle bana lo
  • Username/password ki jagah Personal Access Token use karo
  • Token banane ke liye: GitHub Settings ‚Üí Developer settings ‚Üí Personal access tokens

Deploy to Vercel

Ab Vercel Pe Deploy Karo!

GitHub pe files hai? Perfect! Ab deployment ki baari!

  1. Vercel Account Banao
    Visit: https://vercel.com/signup
    • Click: "Continue with GitHub"
    • Authorize Vercel (give permissions)
    • Ek hi click mein account ban jayega!
  2. New Project Create Karo
    Vercel dashboard pe:
    • Click: "Add New..." ‚Üí "Project"
    • Ya direct visit: https://vercel.com/new
  3. Import GitHub Repository
    • Aapko apni GitHub repositories dikhegi
    • Find: productlens repository
    • Click: "Import"
    üí° Tip: Agar repository nahi dikh rahi, "Adjust GitHub App Permissions" click karo aur access grant karo.
  4. Configure Project
    • Project Name: productlens (ya koi aur naam)
    • Framework Preset: Other (ya blank chhodo)
    • Root Directory: ./ (default)
    • Build Settings: Leave as default
    • Environment Variables: Skip (already in HTML)
  5. Deploy!
    Click: "Deploy" button
    • Vercel build karega aur deploy karega
    • 1-2 minutes wait karo
    • Green checkmark dikha? Success! üéâ
  6. Live URL Mil Gaya!
    Deployment complete hone ke baad:
    • URL milega: https://productlens-xyz123.vercel.app
    • Click karke test karo!
    • Custom domain bhi add kar sakte ho (optional)

    üéä Congratulations!

    Aapka app live hai! üöÄ

Common Issues & Solutions

‚ùå Problem: "Build Failed" on Vercel

Solution:

  • Check ki index.html file hai
  • Verify vercel.json properly uploaded hai
  • Try "Redeploy" button

‚ùå Problem: Images Not Loading

Solution:

  • Check Google Custom Search Engine settings
  • Enable "Image Search" in CSE
  • See google_search_fix.html guide

‚ùå Problem: API Quota Exceeded

Solution:

  • Wait 24 hours for quota reset
  • Or create new Google Cloud project
  • See new_google_api_setup.html

‚ùå Problem: Git Push Failed

Solution:

  • Use GitHub Personal Access Token instead of password
  • Or use GitHub Desktop app (GUI)
  • Or use Method 1 (website upload)
